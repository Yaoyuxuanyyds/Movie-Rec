# RecSys-Movie


## Description
This is a recommendation system for movies based on the ml-25m dataset. The system uses Graph Convolutional Networks (GCN) to recommend movies to users based on their ratings.

## Requirements
- Python 3.7
- PyTorch 1.13.0
- Pandas 1.1.5
- NumPy 1.19.5
- Scikit-learn 0.24.2
- Matplotlib 3.4.3
- TensorboardX 1.8.0
